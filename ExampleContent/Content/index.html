Guppy

Guppy is a tiny ASP.Net application that hosts CommonMark (Markdown) and HTML files from an external location, like a GitHub repository.

Project site »

Content

To add new content to your Guppy site, just add a new CommonMark (.md) or HTML (.html) file to you Guppy Content directory. You can also use directories to create site hierarchies. For example, if you navigate to http://mysite/about, Guppy will look for your content in:

  • Content/about.md
  • Content/about.html
  • Content/about/index.md
  • Content/about/index.html

Theming

The Theme directory is a special directory that exists outside on the Content directory. This directory contains all the files that make up the theme of your website, like _Layout.cshtml, css/js files, and site-wide images. This allows you to easily keep your theme separate from your content. This directory can be accessed via http://mysite/theme/.
